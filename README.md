## Build Setup
* you need to backend with port `3001` first.

``` bash
# install dependencies
yarn install

# serve with hot reload at localhost:8080
yarn serve

# build for production with minification
yarn build

# run unit tests
yarn test:unit

# run lint
yarn lint

# run lint with fix
yarn lint:fix
```
